# **App Name**: Badminton Booking Hub

## Core Features:

- Club Listing: Display a list of badminton clubs (court owners) on the homepage.
- Court Booking: Navigate users from the homepage to a court booking page.
- Calendar Selection: Implement a calendar for users to select the booking date.
- Court Availability Table: Display a scrollable table showing court availability with color-coded statuses (Available, Booked, Blocked, Event) for each 30-minute slot.
- Pricing Information: Show pricing based on time slots and days of the week.
- Booking Summary and Payment: Display selected courts, time slots, and total price, with a button to proceed to booking and payment.
- QR Code Payment: Generate a QR code and payment details message. Allow users to upload proof of payment.
- Booking Management: Allow users to review current bookings.

## Style Guidelines:

- Primary color: #4CAF50 (Green) for a sporty and fresh feel.
- Background color: #F1F8E9 (Very light green).
- Accent color: #FF9800 (Orange) to highlight important actions and elements.
- Font pairing: 'Poppins' (sans-serif) for headlines and 'PT Sans' (sans-serif) for body text.
- Use badminton-themed icons for clarity and visual appeal.
- Mobile-first, responsive layout to ensure usability on all devices.
- Subtle transitions and animations to provide feedback during booking.
